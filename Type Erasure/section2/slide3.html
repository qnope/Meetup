Inheritance — The Downsides

  • Intrusive — every type must inherit from Shape
  • Can't adapt third-party types — you don't own their code
  • Requires pointers — unique_ptr or shared_ptr everywhere
  • No value semantics — can't simply copy a vector<unique_ptr<Shape>>. If shared_ptr, modifying one will contaminates all the others.
  • Pointers invite bugs — dangling, slicing, ownership confusion

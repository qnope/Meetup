void* — "Don't do this"


struct Wrapper {
    void* data;
    void (*draw_fn)(const void*);
};

void draw_circle(const void* p) {
    static_cast<const Circle*>(p)->draw();
}
  • No type safety — cast to the wrong type = undefined behavior
  • No ownership semantics — who deletes what?
  • Compiler can't help you

Never do this in modern C++.

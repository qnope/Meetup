Key Takeaways

  1. The problem is real — heterogeneous collections are everywhere in C++
  2. Inheritance is intrusive — it couples types to a hierarchy
  3. Type erasure = Concept + Model — hide the interface, wrap any conforming type
  4. Value semantics matter — clone pattern gives deep copy without pointers in user code
  5. C++26 brings std::polymorphic — the standard solution for polymorphic value types
